Gmail rank support added.

Dashboard form:
- Player Gmail
- Firebase Auth UID optional
- IGN
- Category
- Tier
- Score
- Region

Main Firebase path:
durbin/userRanksByEmail/<email_key>/ranks/<category>

Example:
durbin/userRanksByEmail/cosa_at_gmail_dot_com/ranks/axe

If UID is also provided, it also writes:
durbin/userRanks/<uid>/ranks/<category>
