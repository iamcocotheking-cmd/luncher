Dashboard loading fix added.

New behavior:
- News loading times out after 12 seconds.
- Tier loading times out after 12 seconds.
- Permission errors show in the UI.
- Tier entries now have Edit and Delete buttons.
- Save errors show as toast instead of staying stuck.

If it still errors:
1. Add admin UID:
   durbin/admins/YOUR_UID = true

2. Paste rules:
   firebase-rules.json

3. Make sure authorized domain is added:
   Firebase Auth -> Settings -> Authorized domains
