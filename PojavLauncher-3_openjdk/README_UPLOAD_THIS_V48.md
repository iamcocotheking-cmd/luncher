# DURBIN Launcher V48 - UI Clean Horizontal Durbin Only

This version fixes the installed APK UI problems reported in screenshots.

Changes:
- Locks launcher UI activities to sensorLandscape.
- Main menu always uses horizontal layout.
- Removes old top-right HyperLauncher tools from home (Tasks/Files/Installer/Settings).
- Removes old home actions: custom controls, install jar, share logs, open files, edit profile.
- Profile/version selection now shows only DURBIN Fabric 1.21.11 and DURBIN Fabric 1.20.1.
- Fabric install screen is restricted to Minecraft 1.21.11 / 1.20.1 and loader 0.19.3 if opened internally.
- Keeps all previous build fixes through V47.

Use this as a full repo replacement. Do not mix with older V47/V46 files.
