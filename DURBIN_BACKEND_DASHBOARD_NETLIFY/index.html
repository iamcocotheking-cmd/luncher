Setup Help

Use this once after deploying to Netlify.

1. Firebase Authorized Domain

Go to Firebase Console → Authentication → Settings → Authorized domains.

Add your Netlify domain like your-site-name.netlify.app.

2. Add yourself as admin

After signing in, copy your UID from the top bar.

In Realtime Database, add:

durbin/admins/YOUR_UID = true

3. Recommended rules

Copy from firebase-rules.json in this ZIP.

4. Netlify deploy

Drag this folder to Netlify, or upload the ZIP to Netlify deploy.
